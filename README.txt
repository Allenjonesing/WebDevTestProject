Starting in index.Bundle1

-Modals are used as login and register forms.
-indexedDBs are used for user credentials and messages posted.
-data is persistant through the use of IDB
-cookies are not used
-CKeditor is used for users to create posts
-Simple images are loaded
-The home page and profile page (index.bundle1.html and messageDatabase.html) are seperate files.
-posts have unique delete buttons

Future updates include:
encryption for security
Remember me button using cookies
security questions for password reset
additional formatting for posts
suggestions for strong passwords
